# SVG PREVIEW
Preview SVG files from any folder

## Installation
Using NPM:
```shell
npm install @eyes22798/svgicon-viewer --global
```

## Usage
In your main.js file:
```shell
# svgicon-viewer <svgFilePath>
svgicon-viewer ./src/assets/svg
```
